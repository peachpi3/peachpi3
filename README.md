## 💻 Web
### 💠 중고거래 플랫폼, 다시마켓
> "단순한 중고 거래를 넘어, 자체 제작된 굿즈를 판매하고 사용자 간의 활발한 소통을 지원하는 통합형 마켓플레이스입니다.<br>
기존 중고거래 사이트들의 복잡한 카테고리를 재정립하기 위해 개발하였습니다."


#### 💁‍♀️ 담당 기능
[프론트엔드]
- 굿즈 목록 조회 페이지 제작
- 커뮤니티 게시판 전 페이지(목록, 상세, 작성, 수정, 삭제) UI 및 사용자 흐름 설계

[백엔드]
- KakaoMap API를 활용해 사용자 선택 위치(법정동)에 따른 거래 기능 구현
- 법정동 정보를 기준으로 한 커뮤니티 게시글 CRUD 기능 개발
- 굿즈 목록 조회 및 아이템 정렬 기능 구현

> 📍 프로젝트 링크: <a href="https://github.com/Team-Nuguri/dashimarket.git">다시마켓</a>

### 💠 도서·영화를 연결해주는 라이프스타일 콘텐츠 플랫폼, BooGiMovie
> "BOOGiMOVIE는 미디어 간의 파편화된 경험을 하나로 묶기 위해 시작되었습니다.<br>
사용자가 하나의 스토리 라인을 따라 도서와 영상을 유기적으로 탐색하고 구매할 수 있는 '통합 미디어 소비 환경'을 구축하고자 개발하였습니다."

[프론트엔드]
- 홈 화면 제작 (베스트 셀러 작품, 인기 도서와 영화), 통합 검색과 인기 검색어 제작
- 마이페이지 내 사용자 프로필과 소장한 영화 화면 제작
- 장바구니 및 결제 완료 화면 제작
- 관리자 로그인 화면 제작
- 관리자 페이지 제작

[백엔드]
- 홈 화면 작품 조회
- 포트원 API를 활용한 도서 & 영화 상품 결제와 장바구니 구현
- JWT 기반 관리자 로그인 구현
- 로그 분석을 통해 인기 검색어 구현 및 연령대별 가장 많이 구매한 작품의 장르 시각화
- 엘라스틱 서치를 활용힌 통합 검색 구현

> 📍 프로젝트 링크: <a href="https://github.com/Team-Oceans/BOOGiMOVIE_ver2.0">부기무비</a>

## 📱Moblie
### 💠 스마트 옷장 원격제어 앱, Pure station
> "패스트 패션 문화로 인한 무분별한 대량생산과 소비를 줄이고 옷에 대한 사회 인식 변화를 목표로, <br>
습도 조절과 살균의 원격제어와 옷을 관리하는 방법을 제시할 수 있는 앱을 구현하고자 하였습니다." <br>

#### 💁‍♀️ 담당 기능
[안드로이드 앱 전반 기능 구현, Firebase 담당]
- 회원가입 및 로그인 기능 설계 및 구현
- Weatherbit REST API를 활용해 사용자 위치 기반 날씨 데이터 수집 및 의류 추천에 활용 가능한 정보로 가공하여 제공
- ESP8266과의 통신을 위해 습도 제어 및 UV 살균 램프 제어 기능 구현
- UV 램프 동작 시간 타이머 및 알림 기능 구현
- Firebase를 활용한 사용자 정보, IoT 상태 데이터 관리 및 실시간 반영

> 📍 프로젝트 링크: <a href="https://github.com/peachpi3/android-PureStation.git">Pure Station</a>
